Mendham Golf & Tennis - Hole 10 Improvement Plan This project is an interactive web presentation designed to showcase the proposed enhancements for the 10th hole at Mendham Golf & Tennis Club. It serves as a digital guide to explain the architectural changes, agronomy improvements, and strategic goals of the 2021 Master Plan. Project Overview The application is built using standard web technologies (HTML, CSS, JavaScript) to ensure compatibility across devices without requiring backend servers or complex frameworks. Features Secure Entry: A simple client-side password protection screen to restrict access to members or stakeholders. Narrative Intro: A multi-step introduction sequence explaining the context of the Master Plan. Interactive Comparison: A customized "Before & After" image slider allowing users to compare existing conditions with the proposed architectural renderings. Hotspot System: Interactive markers (A-F) placed on the map that trigger detailed pop-ups with renderings and descriptions of specific features (e.g., Fairway Bunkers, Green Surrounds). Responsive Design: Fully optimized for desktop, tablet, and mobile viewing. File Structure index.html: The main entry point. Contains the Login screen, Introduction sequence, and the Interactive Map/Slider logic. images/: (Assumed) Folder containing the aerials (Aerial_PR.png, Aerial_EX.png) and renderings. How to Use Clone or download this repository. Ensure all images are placed in the root directory or updated in the src paths in the HTML. Open index.html in any modern web browser. Login Password: password (Default setting in the script). Technologies Used HTML5 for structure. CSS3 for styling, animations, and the sliding comparison layout. Vanilla JavaScript for logic (login, slider movement, hotspot interactions). Credits Master Plan: Robert McNeil, ASGCA (Northeast Golf Company). Contractors: NMP Golf.
